# Project Overview

`financial-statement-analysis` is the front-of-stack borrower financial analysis repo in the public commercial credit-risk portfolio.

## Portfolio role

It standardises borrower statements, ratios, trends, and working-capital metrics into reusable borrower analysis outputs.

## Upstream inputs

- synthetic borrower statement packs and benchmark reference inputs staged under `data/`

## Downstream consumers

- `PD-and-scorecard-commercial`
- `LGD-commercial`
